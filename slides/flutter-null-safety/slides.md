# Flutter Null Safety

Apa itu Null Safety dan kenapa penting?

---
# Masalah sebelum Null Safety

- NullPointerException
- Crash yang sulit dilacak
- Kode defensif berlebihan

---
# Setelah Null Safety

- `String?` → nullable
- `String` → non-nullable
- `late` → diinisialisasi nanti

---
# Kesimpulan

Null Safety membuat kode Dart lebih aman dan prediktif.
