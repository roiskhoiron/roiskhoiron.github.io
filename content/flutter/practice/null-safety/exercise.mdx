# Latihan Null Safety

## Level 1: Mudah 🟢

Buat fungsi yang menerima `String?` dan mengembalikan panjang string, atau 0 kalau null.

```dart
int getLength(String? text) {
  // Tulis kode kamu di sini
}
```

**Test:**
- `getLength("CodingSkuy")` → 10
- `getLength(null)` → 0

---

## Level 2: Menengah 🟡

Buat class `User` dengan field nullable `email`. Buat method `getEmailDisplay()`:

- Kalau email ada → tampilkan email
- Kalau null → tampilkan "Email belum diisi"

---

## Level 3: Tantangan 🔴

Dari list `String?` berikut, filter yang **bukan null** lalu gabung dengan koma:

```dart
List<String?> items = ["Flutter", null, "Dart", null, "CodingSkuy"];
// Hasil: "Flutter, Dart, CodingSkuy"
```

**Clue:** Pakai `.whereType<String>()` atau `.where((x) => x != null)` + `!` atau `??`.
