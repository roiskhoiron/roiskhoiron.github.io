# Praktek Null Safety

Di sesi ini kamu bakal **langsung nulis kode** dan lihat sendiri gimana Null Safety bekerja.

## Yang Kamu Butuhin

1. **DartPad** — buka [dartpad.dev](https://dartpad.dev) (gak perlu install apa-apa!)
2. Atau **VS Code** + ekstensi Dart
3. Semangat belajar! 💪

## Persiapan

Buka DartPad, hapus semua kode yang ada, dan ketik ulang kode berikut:

```dart
void main() {
  print("Selamat datang di praktek Null Safety!");
}
```

Klik **Run** — kalau muncul "Selamat datang di praktek Null Safety!", kamu siap lanjut. 🎉
