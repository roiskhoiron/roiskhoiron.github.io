# Project: Form Validator Mini

Bikin fungsi validasi form yang ngecek beberapa field nullable.

## Spesifikasi

Buat fungsi `validateForm` yang menerima `Map<String, String?>` dan mengembalikan `Map<String, String>` (error messages):

```dart
Map<String, String> validateForm(Map<String, String?> form) {
  // Return map: field_name → error_message
  // - "name" wajib diisi
  // - "email" wajib diisi dan harus mengandung "@"
  // - "age" wajib diisi dan harus angka
}
```

**Contoh input:**
```dart
{
  "name": null,
  "email": "rois",
  "age": "abc"
}
```

**Output yang diharapkan:**
```dart
{
  "name": "Nama wajib diisi",
  "email": "Email harus mengandung @",
  "age": "Umur harus angka"
}
```

## Challenge Tambahan

- Tambahkan validasi `phone` — minimal 10 digit
- Tampilkan pesan error yang user-friendly (pakai bahasa Indonesia!)
