## Expected Output

```
Budi — 0812xxxx | budi@mail.com
Siti — (tidak ada telepon) | siti@mail.com
Rudi — 0878xxxx | (tidak ada email)
Total: 3 kontak
```

## Challenge Tambahan 🔥

Tambah method `cariKontak(String? query)`:

- Kalau `query` null → tampilkan semua
- Kalau `query` ada → cari yang `nama` mengandung `query`
- **Pake null-aware operator!**
