# 🎓 Tutorial Null Safety

## Introduction

Di tutorial ini, kamu bakal **langsung praktek** bikin aplikasi Flutter yang pake Null Safety.

**Yang bakal kamu pelajari:**
- Cara enable Null Safety di project Flutter
- Perbedaan `String` dan `String?`
- Pake operator `?.` dan `??`
- Pake `late` keyword
- Migrasi kode lama ke Null Safety

---

## Prasyarat

Sebelum mulai, pastikan:

- ✅ Flutter SDK ≥ 2.12 terinstall
- ✅ Dart ≥ 2.12
- ✅ Udah baca [Artikel Null Safety](/articles/flutter/null-safety/)
- ✅ Udah coba [Playground](/articles/flutter/null-safety/playground)

> Cek versi Flutter: `flutter --version` — pastikan **3.x** ya!
