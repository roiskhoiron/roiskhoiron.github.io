# Part 2: Class & Late

Bikin file `lib/user.dart`:

```dart
class User {
  final String nama;       // Wajib — gak boleh null
  final String? email;     // Boleh null
  final int umur;          // Wajib
  String? bio;             // Boleh null, bisa diubah

  // Constructor
  User({
    required this.nama,
    this.email,
    required this.umur,
  });

  String get info {
    final emailTampil = email ?? '(tidak ada email)';
    final bioTampil = bio ?? '(belum ada bio)';
    return '$nama ($umur) — $emailTampil — $bioTampil';
  }
}
```

## Late Keyword

Kalau ada variable yang **pasti** diisi nanti:

```dart
class DatabaseService {
  late Database _db;

  Future<void> init() async {
    _db = await Database.connect(); // diisi async
  }

  Future<List<User>> getUsers() async {
    return _db.query('SELECT * FROM users');
  }
}
```

⚠️ **Peringatan:** Panggil method `init()` dulu sebelum `getUsers()` — error kalau `_db` belum diisi.

## Gabungin Semua

```dart
import 'user.dart';

void main() {
  // Semua field required terisi — aman
  final user1 = User(nama: 'Budi', umur: 17, email: 'budi@mail.com');
  final user2 = User(nama: 'Siti', umur: 15);

  print(user1.info);
  print(user2.info);
}
```
