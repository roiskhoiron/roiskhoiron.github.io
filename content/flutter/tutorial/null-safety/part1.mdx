# Part 1: Basic Types

## Bikin Project Baru

```bash
flutter create null_safety_demo
cd null_safety_demo
```

## Cek Null Safety Status

Buka `pubspec.yaml` dan pastikan:

```yaml
environment:
  sdk: ">=2.17.0 <3.0.0"
```

Buat file `bin/null_safety.dart`:

```dart
void main() {
  // String — wajib diisi
  String nama = 'Budi';
  print(nama.length); // 4

  // String? — boleh null
  String? namaNullable;
  print(namaNullable?.length); // null (gak crash!)
  print(namaNullable?.length ?? 0); // 0

  // int — wajib diisi
  int umur = 17;
  print(umur);

  // int? — boleh null
  int? umurNullable;
  print(umurNullable ?? 'belum diisi');
}
```

## Yang Perlu Diingat

| Kode | Arti |
|---|---|
| `String nama` | `nama` WAJIB punya value |
| `String? nama` | `nama` BOLEH null |
| `nama?.length` | Kalau null, balikin null |
| `nama ?? 'x'` | Kalau null, pakai default |
