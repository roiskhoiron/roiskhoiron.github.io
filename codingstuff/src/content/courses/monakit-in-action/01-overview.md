## What is MonaKit?

A **multi-format content platform** built with Astro.

> Ship knowledge cards, articles, presentations, and announcements — all from Markdown.

---

## Four Content Types

| Type | Purpose |
|------|---------|
| **Cards** | Bite-sized research summaries |
| **Blogs** | Long-form articles |
| **Slides** | Reveal.js presentations |
| **Doodles** | Announcements & release notes |

---

## Plus Courses

Structured learning with chapters, parts, slide chapters, and optional video via Cloudflare Stream.

---

## Quick Start

```bash
pnpm create astro@latest my-site -- --template monakit/monakit

cd my-site && pnpm install
cp .env.example .env
pnpm dev
```
