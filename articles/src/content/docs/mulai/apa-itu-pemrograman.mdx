# Apa Itu Pemrograman?

**Pemrograman** (atau *coding*) adalah cara kita memberikan instruksi ke komputer untuk melakukan sesuatu.

Bayangin kamu lagi ngasih instruksi ke temen buat masak mie instan:

1. Buka bungkus mie
2. Rebus air
3. Masukkan mie
4. Tiriskan
5. Tambahkan bumbu

Nah, coding itu sama persis. Kamu ngasih instruksi langkah demi langkah ke komputer. Bedanya, komputer itu **kaku** — kalau instruksinya salah, hasilnya juga salah.

## Kenapa Coding Penting?

- 🧠 **Melatih logika** — berpikir step-by-step
- 🔧 **Problem solving** — belajar mecahin masalah besar jadi kecil-kecil
- 🎨 **Kreativitas** — kamu bisa bikin game, apps, website sendiri
- 💼 **Karir masa depan** — semua industri butuh programmer

## Bahasa Pemrograman Populer

| Bahasa | Gampang | Buat Apa |
|---|---|---|
| Scratch | 🟢 Sangat | Anak-anak, game sederhana |
| Python | 🟢 Mudah | AI, data, web, pemula |
| JavaScript | 🟡 Sedang | Website, games |
| Dart/Flutter | 🟡 Sedang | Aplikasi HP |
| Kotlin | 🔴 Sulit | Aplikasi Android |

## Mulai dari Mana?

Kalau kamu **pemula banget**:
1. 🧒 **SD (6-9 thn)** → ScratchJr atau Scratch
2. 🧒 **SD (10-12 thn)** → Scratch + Python dasar
3. 🧑 **SMP (13-15 thn)** → Python atau Flutter

> **Ingat:** Semua programmer hebat dulunya juga pemula. Kuncinya: **konsisten** dan **gak takut salah**!
