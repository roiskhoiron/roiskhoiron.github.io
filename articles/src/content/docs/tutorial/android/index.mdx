# 📱 Tutorial Android

Mau bikin aplikasi yang jalan di **HP Android**? Kamu ada di tempat yang tepat!

**Android** adalah sistem operasi HP paling populer di Indonesia. Bayangin — aplikasi yang kamu buat bisa dipake jutaan orang!

---

## 🔧 **Yang Kamu Butuhin**

- **Android Studio** — software buat bikin aplikasi Android (gratis!)
- **Bahasa Kotlin** — bahasa modern buat Android (mirip Python + JavaScript)
- Atau **Java** — bahasa klasik yang masih banyak dipake
