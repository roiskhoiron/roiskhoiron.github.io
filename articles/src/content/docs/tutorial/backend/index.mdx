# ⚙️ Tutorial Backend

Pernah mikir **gimana caranya** aplikasi kayak Instagram atau TikTok nyimpen semua foto dan video?

Jawabannya: **Backend**! 🖥️

Backend adalah bagian aplikasi yang **gak keliatan** — yang kerja di server, nyimpen data, dan ngatur semuanya.

---

## 🧠 **Yang Bakal Kamu Pelajari**

- **Server** — komputer yang selalu nyala buat melayani pengguna
- **API** — jembatan antara aplikasi HP dan server
- **Database** — tempat nyimpen data (user, foto, video, dll)
- **Authentication** — sistem login dan registrasi

---
