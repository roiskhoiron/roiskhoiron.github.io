# 🐍 Tutorial Python

**Python** itu bahasa coding yang paling **gampang dibaca** — mirip bahasa Inggris biasa!

Coba lihat:

```python
nama = input("Siapa nama kamu? ")
print(f"Halo {nama}! Selamat datang di Python! 🎉")
```

**Keren kan?** Hanya 2 baris dan kamu udah bisa bikin program ngomong sama kamu!

---

## 🎯 **Yang Akan Kamu Pelajari**

- **Variable** — nyimpen data (angka, teks, gambar)
- **Condition** — "kalau ini, lakukan itu" (if/else)
- **Loop** — ngulang sesuatu berkali-kali (for/while)
- **Function** — bikin blok kode sendiri yang bisa dipake ulang
- **Project** — bikin program beneran!
