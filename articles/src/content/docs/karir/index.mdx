## 💡 **Tips dari Kakak‑kakak Programmer**

1. **Mulai dari yang kecil** — jangan langsung pengen bikin game kayak Mobile Legends
2. **Praktek tiap hari** — 15 menit sehari lebih baik dari 5 jam seminggu sekali
3. **Bertanya itu boleh** — semua programmer pernah ngalamin error
4. **Tunjukkin karyamu** — bikin portofolio di GitHub
5. **Nikmati prosesnya** — coding itu seru, bukan beban!
