# Mobile Engineer

> *"Aku mulai bikin aplikasi HP karena penasaran. Ternyata jadi mobile engineer itu lebih dari sekadar nulis kode."*

Mobile engineer — profesi yang menggabungkan desain, logic, dan pengalaman user dalam satu paket.

**Awal mula:**
Aku mulai mobile dengan Flutter 2021. Pertama bikin todo app. Kedua bikin counter app. Ketiga — masih counter app. Tapi pelan-pelan, aku mulai bikin aplikasi beneran.

**Apa yang membedakan mobile engineer:**

1. **Platform knowledge** — kamu harus tahu Android dan iOS. Bukan cuma nulis kode, tapi paham lifecycle, constraints, dan quirk masing-masing.
2. **State management** — aplikasi mobile itu stateful. Layar rotating, app di-minimize, notifikasi masuk — semua harus dihandle.
3. **UX sensitivity** — mobile itu layar kecil. Setiap pixel berharga. Loading state, error state, empty state — semua harus dipikirkan.
4. **Offline first** — koneksi putus di tengah jalan? Aplikasi harus tetap jalan.

**Ceritaku:**
Dari Flutter experimentation sampe CodingSkuy apps, perjalanannya panjang. Yang paling kubanggakan: aplikasi yang benar-benar dipake orang.

> *"Mobile engineer bukan cuma 'bikin aplikasi'. Tapi bikin pengalaman yang bisa dibawa ke mana-mana."*
