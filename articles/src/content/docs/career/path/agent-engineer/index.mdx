# Agent Engineer

> *"Kalau AI engineer bikin model yang jawab pertanyaan, agent engineer bikin AI yang *ngerjain sesuatu*."*

Agent engineer — evolusi dari AI engineer. Bukan cuma bikin chatbot, tapi bikin sistem AI yang bisa pakai tools, ambil keputusan, dan ngerjain task kompleks.

**Apa yang dilakukan agent engineer:**

1. **Tool design** — agent butuh tools: search, database, API. Tool harus dirancang biar agent bisa pake dengan efektif.
2. **Orchestration** — mana agent yang ngapain? Urutannya gimana? Kalau error, fallback ke mana?
3. **Memory & context** — agent harus ingat konteks. Tapi konteks terbatas. Gimana manage-nya?
4. **Safety & guardrails** — agent bisa ngelakuin hal yang tidak diinginkan. Bagaimana mencegahnya?

**Ceritaku:**
Eksperimen bikin agent buat bantu nulis artikel CodingSkuy. Agent riset → agent nulis → agent review. Masing-masing punya tools sendiri. Hasilnya lebih efisien, tapi kompleksitas koordinasinya tinggi.

> *"Agent engineer adalah yang bikin AI tidak cuma ngomong, tapi kerja."*
