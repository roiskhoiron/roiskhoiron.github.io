# AI Engineer

> *"AI engineer itu bukan 'panggil API ChatGPT'. Bukan juga 'training model dari nol'. Di antara keduanya."*

AI engineer — profesi yang naik drastis sejak 2023. Tapi banyak yang salah kaprah tentang apa itu.

**Yang dilakukan AI engineer sebenarnya:**

1. **Problem framing** — masalah apa yang bisa diselesaikan AI? Lebih penting: masalah apa yang *tidak* perlu AI?
2. **Data pipeline** — ngumpulin, bersihin, label data. Ini 80% kerjaan.
3. **Model selection & evaluation** — open source vs closed, RAG vs fine tuning, murah vs mahal.
4. **Deployment & monitoring** — model di production, performa turun, data drift.
5. **Prompt engineering** — untuk LLM, cara nulis prompt menentukan 90% hasil.

**Ceritaku:**
Eksperimen AI untuk CodingSkuy — dari ChatGPT buat bantu nulis, sampe RAG pipeline yang paham semua artikel. Setiap langkah ada pelajaran baru.

**Tips untuk mulai:**
- Jangan mulai dari training model. Mulai dari pake API existing.
- Pahami prompt engineering dulu.
- Pelajari RAG — itu pintu masuk yang paling praktis.

> *"AI engineer adalah yang tahu kapan pakai AI dan kapan cukup pakai IF-ELSE."*
