# ATS Friendly Resume

> *"Resume keren hasil desain Canva — tapi tidak lolos ATS. Resume plain text — malah dipanggil."*

ATS (Applicant Tracking System) adalah software yang baca resume sebelum manusia. Dan ATS itu buta — dia tidak lihat desain, cuma lihat teks.

**Yang bikin resume gagal ATS:**

1. **Format kompleks** — kolom, tabel, graphic element. ATS bacanya berantakan.
2. **File gambar** — JPG, PNG, atau PDF yang ternyata gambar (bukan teks).
3. **Singkatan tidak jelas** — "Flutter, BLoC, Riverpod" tanpa kepanjangan.
4. **Posisi tidak jelas** — ATS cari keyword "Flutter Developer" tapi resume-nya "Software Engineer."

**Tips ATS friendly:**

1. **Pakai format standar** — section jelas: Experience, Education, Skills.
2. **Keyword dari job description** — kalau JD bilang "Riverpod", tulis "Riverpod" bukan "state management."
3. **Bold dan italic aman** — TAPI jangan pake tabel atau column.
4. **Simpan sebagai PDF teks** — bukan PDF gambar.
5. **Cek dengan tool ATS simulator** — before applying.

> *"ATS bukan musuh. ATS adalah filter. Buat resume yang lolos filter dulu, baru desain bisa menyusul."*
