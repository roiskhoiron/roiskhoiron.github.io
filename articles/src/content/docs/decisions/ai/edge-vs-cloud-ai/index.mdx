# Edge AI vs Cloud AI

> *"Cloud AI itu pintar, tapi lambat karena harus kirim data bolak-balik. Edge AI itu cepat, tapi modelnya lebih kecil."*

**Cloud AI:**
- Model besar dan pintar (GPT-4, Claude 3.5)
- Komputasi di server — perangkat hanya kirim request
- Dependen internet — tanpa koneksi, mati
- Latensi lebih tinggi (100ms-2s)
- Privasi — data dikirim ke cloud

**Edge AI (on-device):**
- Model kecil yang jalan di perangkat
- Latensi sangat rendah (ms)
- Bisa offline — model tetap jalan
- Privasi — data tidak ke mana-mana
- Tapi: model lebih terbatas, komputasi terbatas, baterai terpengaruh

**Ceritaku:**
Coba eksperimen: bikin classifier sederhana untuk kategorisasi konten.

Versi cloud: pake GPT-4 via API. Akurat, tapi tiap klasifikasi butuh 500ms + koneksi.

Versi edge: pake TensorFlow Lite di HP. Training butuh effort, akurasi lebih rendah, tapi klasifikasi selesai dalam 50ms, tanpa internet.

> *"Cloud AI untuk: kecerdasan tinggi, update model mudah, no device constraint. Edge AI untuk: offline, latensi kritis, privasi ketat, biaya API besar. Keduanya bisa digabung: edge untuk real-time, cloud untuk fallback."*
